3 · State Manager (as implemented)

The state machine is split across three automations plus one script — not a single automation as the design diagram implies. pergola_state_manager handles rules 1–4 (frost/rain/rain_stopped/user_override) and the default delegation to script.pergola_evaluate_state. not_enough_sun entry and exit each have a dedicated automation because they have distinct trigger/condition patterns (the sensor's built-in delays make extra for: guards unnecessary). Only these three automations (plus the script) may write input_select.pergola_automation_state.

4 · Evaluate State Script (as implemented)

script.pergola_evaluate_state (mode: single, max_exceeded: silent) evaluates rules 5–8 and writes input_select.pergola_automation_state. Called from two places: the pergola_state_manager default branch, and the post-rain recovery script final step (Step 4 — not yet implemented). This is the single place for lower-priority rule logic.

5 · Pending: Cover Response + Tilt Calculation (Step 3)

Not yet implemented. Step 3 adds:

  • automation.pergola_cover_response — triggers on state change + every 5 min; checks automatic_enabled; computes slat_angle per state; applies deadband; calls script
  • script.pergola_set_slat_angle — accepts slat_angle (float, degrees); applies hardware correction (7/5.5/0.5 offsets); clamps to [0,100]; sends to both covers
  • Fills in template sensor formulas for sensor.pergola_effective_sun_angle, sensor.pergola_slat_angle, sensor.pergola_tilt_position (stubs currently return unavailable)
  • Cooling formula: A_eff → MaxOpenEast/MaxOpenWest geometry → slat_angle (no_sun/not_enough_sun → slat_angle=90°; heating stub → slat_angle=90° until Step 5)
See design: Section 3 Cover Response and design: Section 4 Tilt Calculation.

6 · Pending: Post-Rain Recovery Script (Step 4)

Not yet implemented. Step 4 adds script.pergola_post_rain_recovery. When the state machine enters rain_stopped, this script runs the stepped drain sequence: slat_angle 8° → wait 5 min → 15° → wait 5 min → check hourly_rain → [wait 30 min if still wet] → call script.pergola_evaluate_state to exit rain_stopped. Calls script.pergola_set_slat_angle for cover movement (no deadband check — drain must complete).
Depends on Step 3 (script.pergola_set_slat_angle must exist first). See design: Section 5 Post-Rain Recovery.
